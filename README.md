PasswordStrengthAnalyzer
========================

PasswordStrengthAnalyzer analyzes given password and scores its strength from 0 to 100%.

Usage
-----

    $analyzer = new Yarri\PasswordStrengthAnalyzer();
    $score = $analyzer->analyze("someFAIRpasswd");
